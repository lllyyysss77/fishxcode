# 账户注册与管理

> 已有账号？直接查看 [快速开始](/start) 配置工具。
> 想了解平台全貌？请查看 [平台导航](/platform)。

## 一、获取 API Key

### 1. 注册账号

访问 [fishxcode.com](https://fishxcode.com/register?aff=9CTW)，点击**注册**：

![FishXCode 首页](/img/start/api-01-home.png)

选择注册方式（GitHub、LinuxDO 或用户名注册）：

![注册方式选择](/img/start/api-02-register.png)

填写用户名、密码和确认密码完成注册：

![填写注册信息](/img/start/api-03-register-form.png)

### 2. 登录

注册完成后使用用户名和密码登录：

![登录页面](/img/start/api-04-login.png)

登录后进入控制台：

![控制台首页](/img/start/api-05-console.png)

### 3. 创建令牌

进入**控制台 → 令牌管理 → 添加令牌**，填写表单：

![添加令牌](/img/start/api-06-token-create.png)

- **令牌分组**建议选择 **官渠**，该分组包含全系列 Claude 模型，会根据任务复杂度自动选择，无需手动切换。具体渠道和模型以官网实际为准，请前往 [定价页](https://fishxcode.com/pricing) 查看最新信息。

创建完成后在令牌列表点击**复制**按钮获取 API Key（格式为 `sk-xxx`）：

![复制令牌](/img/start/api-07-token-copy.png)

### 4. 充值

进入**控制台 → 钱包管理**，支持支付宝、微信或兑换码：

![充值页面](/img/start/api-08-wallet.png)

| 方式 | 操作路径 |
|---|---|
| 支付宝 | 输入/选择金额 → 支付宝 |
| 微信 | 输入/选择金额 → 微信 |
| 兑换码 | 兑换码区域输入 → 点击兑换额度 |

::: tip 倍率说明
分组名称中的 `0.5x倍率` 表示消费倍率，数值越小获得更多 Token，如 `0.5x` 即 1 元可使用官方 1000 万 Token（官方原价 500 万）。
:::

## 二、令牌管理

### 令牌分组说明

创建令牌时需要选择分组，不同分组决定了可用的模型范围和消费倍率：

| 分组类型 | 说明 | 适用场景 |
|---|---|---|
| **官渠** | 包含全系列 Claude 模型，自动路由 | 推荐大多数用户使用 |
| **低倍率分组** | 消费倍率更低，性价比更高 | 高频调用、成本敏感场景 |

::: tip 推荐
如果你不确定选哪个分组，直接选择 **官渠** 即可。系统会根据任务复杂度自动选择最合适的模型。
:::

### 令牌安全建议

- **定期轮换**：建议每 30-90 天更换一次令牌，降低泄露风险
- **不要明文存储**：避免将 API Key 直接写入代码或提交到 Git 仓库，推荐使用环境变量
- **权限最小化**：为不同项目创建独立的令牌，便于管理和追踪
- **泄露处理**：如果怀疑令牌泄露，立即在控制台删除该令牌并创建新的

### 修改与删除令牌

在**控制台 → 令牌管理**页面：

- **查看详情**：点击令牌名称查看分组、模型范围和创建时间
- **复制令牌**：点击列表中的复制按钮
- **删除令牌**：点击删除按钮，删除后该令牌立即失效，使用该令牌的工具将无法继续调用

::: warning 注意
删除操作不可撤销，请确认不再使用后再删除。
:::

## 三、账户安全

### 密码与登录安全

- 使用**强密码**（8 位以上，包含大小写字母和数字）
- 不要在多个平台使用相同密码
- 如忘记密码，可在登录页面通过找回密码流程重置

### 第三方登录绑定

FishXCode 支持以下第三方登录方式：

| 方式 | 说明 |
|---|---|
| **GitHub** | 推荐开发者使用，一键授权登录 |
| **LinuxDO** | 社区用户可直接登录 |

绑定第三方账号后，可直接使用对应平台授权登录，无需输入密码。

### 异常检测与处理

如果你发现以下异常情况，建议立即采取措施：

1. **消费异常增加** → 检查令牌是否泄露，必要时删除所有令牌并重新创建
2. **无法登录** → 尝试找回密码；如仍无法解决，联系官方支持
3. **余额异常扣减** → 在使用日志中排查具体调用记录，确认是否为正常消耗

::: tip 联系支持
遇到账户异常问题，请通过 [联系我们](https://fishxcode.com/contact) 页面获取帮助。
:::

## 四、常见问题

### 注册时收不到验证码？

1. 检查邮箱是否填写正确（注意拼写和域名）
2. 查看垃圾邮件/垃圾箱文件夹
3. 等待 1-2 分钟后重试，避免频繁请求
4. 如仍无法收到，尝试更换邮箱或使用第三方登录

### 令牌创建后在哪里查看？

进入 **控制台 → 令牌管理**，已创建的令牌会在列表中显示。点击复制按钮即可获取 `sk-xxx` 格式的 API Key。

### 充值未到账怎么办？

1. 确认支付是否成功（查看支付宝/微信的交易记录）
2. 等待 1-5 分钟，系统可能存在短暂延迟
3. 刷新控制台钱包页面
4. 如超过 10 分钟仍未到账，请联系官方客服并提供支付截图

### 如何查看消费明细？

进入 **控制台 → 使用日志**，可查看每次 API 调用的：

- 调用时间和模型名称
- Token 消耗量和费用
- 请求 IP 和详细记录
